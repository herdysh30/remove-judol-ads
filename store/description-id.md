# Remove Judol Ads — Deskripsi Chrome Web Store (ID)

## Nama

Remove Judol Ads

## Deskripsi Singkat (132 karakter max)

Blokir iklan judi online (judol) secara otomatis. Hapus banner, popup, overlay, dan redirect ke situs judi dari semua website.

## Deskripsi Lengkap

🛡️ Remove Judol Ads — Ekstensi pemblokir iklan judi online untuk Chrome

Bersihkan browsing kamu dari iklan judi online yang mengganggu! Ekstensi ini otomatis mendeteksi dan menghapus berbagai jenis iklan judol dari website yang kamu kunjungi.

✨ FITUR UTAMA:

🔍 Deteksi Otomatis
• Blokir link iklan judi berdasarkan keyword, TLD, dan domain
• Hapus overlay dan popup iklan yang menutupi halaman
• Cegah redirect dan popunder ke situs judi online

🎯 Element Picker
• Pilih elemen iklan langsung di halaman web
• URL otomatis di-extract dan ditambahkan ke blocklist
• Semudah point-and-click!

⚙️ Customizable
• Tambah/hapus keyword filter sendiri
• Kelola TLD yang diblokir (.xyz, .click, .fun, dll)
• Atur domain gambar dan URL shortener

📊 Statistik
• Lihat berapa iklan yang berhasil diblokir hari ini
• Pantau total aturan yang aktif

🚀 Ringan & Cepat
• Tanpa dampak signifikan ke performa browsing
• CSS hiding sebelum halaman render — iklan tidak sempat muncul
• Manifest V3 compliant

💡 CARA PAKAI:

1. Install ekstensi
2. Klik ikon 🛡️ di toolbar
3. Ekstensi langsung aktif! Iklan judol otomatis diblokir
4. Gunakan Element Picker untuk memblokir iklan yang belum terdeteksi

🔒 PRIVASI:
• Tidak mengumpulkan data pribadi
• Tidak mengirim data ke server manapun
• Semua pemrosesan dilakukan secara lokal di browser kamu
