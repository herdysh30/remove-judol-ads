## Full Description

🛡️ Remove Judol Ads — Online Gambling Ad Blocker for Chrome

Clean your browsing experience from annoying online gambling ads! This extension automatically detects and removes various types of gambling advertisements from websites you visit.

✨ KEY FEATURES:

🔍 Automatic Detection
• Block gambling ad links based on keywords, TLDs, and domains
• Remove overlays and popups that cover page content
• Prevent redirects and popunders to gambling sites

🎯 Element Picker
• Select ad elements directly on the webpage
• URLs are automatically extracted and added to the blocklist
• As easy as point-and-click!

⚙️ Customizable
• Add/remove your own keyword filters
• Manage blocked TLDs (.xyz, .click, .fun, etc.)
• Configure image domains and URL shorteners

📊 Statistics
• See how many ads were blocked today
• Monitor total active rules

🚀 Lightweight & Fast
• No significant impact on browsing performance
• CSS hiding before page render — ads don't even flash
• Manifest V3 compliant

💡 HOW TO USE:

1. Install the extension
2. Click the 🛡️ icon in the toolbar
3. Extension is active immediately! Gambling ads are blocked automatically
4. Use Element Picker to block ads that aren't detected yet

🔒 PRIVACY:
• Does not collect personal data
• Does not send data to any server
• All processing is done locally in your browser
